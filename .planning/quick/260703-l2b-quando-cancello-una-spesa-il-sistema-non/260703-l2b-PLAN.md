# Quick Task: Conferma cancellazione spese/transazioni collegate

## Goal

Quando l'utente elimina spese o transazioni, proporre opzionalmente la rimozione delle entità collegate:

- **Spesa (singola + bulk):** checkbox per eliminare anche le transazioni collegate
- **Transazione (singola + bulk):** se rapporto 1:1 con la spesa (`transactionCount === 1`), checkbox per eliminare anche la spesa

## Tasks

### Task 1: Service + validations + actions

- `lib/services/expense-deletion.ts` — `deleteExpensesWithOptions({ deleteLinkedTransactions })`
- Extend `deleteTransactionsAndReconcileExpenses` con `deleteLinkedExpenses` per spese 1:1
- Schemi Zod: flag booleani su delete expense/transaction (singolo + bulk)
- Aggiornare `lib/actions/expenses.ts` e `lib/actions/transactions.ts`
- Esporre `transactionCount` in `ExpenseRow` (DAL)

### Task 2: UI dialogs

- `DeleteExpenseMenuItem` + `BulkDeleteExpensesDialog` — checkbox transazioni collegate (sempre visibile se count > 0)
- `DeleteTransactionMenuItem` + `BulkDeleteTransactionsDialog` — checkbox spesa 1:1 quando applicabile
- Passare metadati necessari dalla tabella (transactionCount, expenseTransactionCount, expenseTitle)

### Task 3: Tests

- Unit test per `expense-deletion` e flag `deleteLinkedExpenses` su transaction-deletion
