# Quick Task 260709-kp1: Structural balance reading on the Bilancio KPI card

## Goal (decision B+, discussed 2026-07-09)

The Bilancio card today reads `totalIn − totalOut` where totalIn includes
`income_extraordinary` — a one-off inheritance/bonus produces "Spendi meno di quanto
guadagni ✓" while the recurring budget is in deficit. Evolve the **reading** (not the
headline number) to be structural-aware and carry the evidence:

| totale | strutturale | reading | sentiment |
|---|---|---|---|
| > 0 | ≥ 0 | "Spendi meno di quanto guadagni" | good |
| > 0 | < 0 | "Senza le entrate straordinarie saresti a −X €" | warn |
| < 0 | any | "Spendi più di quanto guadagni" | bad |
| = 0 | any | "Sei in pareggio" | neutral |

Headline value stays `totalIn − totalOut` (reconciles with the Entrate/Uscite cards).
Structural balance = recurring income only (`nature.code = 'income'`) − totalOut.
Domain basis: `income_extraordinary` is only new non-recurring money (refunds net in OUT,
divestments are negative allocation — CONTEXT.md), so excluding it is semantically sound.

## Tasks

### 1. DAL: recurring-income total + structural balance
- `getOverviewAmountTotals` (lib/dal/dashboard.ts): add
  `totalInRecurring: coalesce(sum(case when direction.code='in' and nature.code='income' …))::text`
  — nature is already joined (override-aware).
- `OverviewAggregateRow`: add **optional** `totalInRecurring?: string | null`
  (existing test literals compile unchanged; absent → unknown).
- `buildOverviewData`: add `structuralBalance: string | null` to `OverviewData` —
  `totalInRecurring − totalOut` when the field is present, else `null`. No delta
  (B+ shows no trend for structural). Update both catch-fallback literals
  (dashboard.ts getOverviewAmountTotals + overview.ts getOverview) to include the field
  so real callers always produce a non-null structural balance.

### 2. KPI card reading (components/dashboard/overview/kpi-row.tsx)
- `balanceReading(balance: number, structural: number | null)` per the matrix above;
  `structural === null` → legacy behavior. Warn text uses `formatEur(structural)`.

### 3. Tests
- dashboard-dal: `buildOverviewData` propagates structuralBalance (present / absent /
  extraordinary-heavy fixture where balance > 0 but structural < 0).
- kpi-row: `balanceReading` matrix incl. the warn case carrying the formatted amount,
  and null-structural fallback.
