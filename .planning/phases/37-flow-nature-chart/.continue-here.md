## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| SubagentNoBash | gsd-executor subagents lack Bash permissions — they can write files but cannot run git commit, yarn test, drizzle-kit, or any shell command. Session was wasted on two agents that both failed silently. | blocking | Always pass `--interactive` to /gsd-execute-phase. Do NOT spawn parallel worktree agents without verifying Bash is allowed in subagent context. |
