| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |

| T-68-01 | Tampering / Information Disclosure | `tagId`/`?tag=` on `getTransactions` and (starting Wave 3) every dashboard DAL function | medium | mitigate | Structural: `tagScopedTransactions()` only ever narrows rows already scoped by `eq(transaction.userId, userId)` in the same query's WHERE clause — a foreign `tagId` matches zero rows, no leak possible.
