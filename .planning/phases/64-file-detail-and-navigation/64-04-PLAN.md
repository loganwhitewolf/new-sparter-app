<objective>
Close out DET-09's navigation wiring: make the Import table's file name a real link to
`/import/[fileId]` (D-04) with a "Dettagli" menu entry (D-06), and repoint every remaining
`/import?fileId=…` cross-reference — in the Transactions table's file column and both Phase 63
detail-page clients — to `importFileDetailHref` (D-05, and Phase 63's D-16 carried forward).

Purpose: this is the last piece making the file a first-class linked entity everywhere it is
referenced, not just from its own list. 64-RESEARCH.md flags this exact repoint as "Pitfall 3 —
easy to defer, but the milestone goal is uniform cross-references."

Output: `ImportDisplayNameEdit` gains an optional link mode; `ImportRowActions` gains a
Dettagli entry; three existing `?fileId=` hrefs become `importFileDetailHref(...)` calls.

Note: this plan touches 9 files across 2 tasks, but the Task 2 repoints are mechanical
one-line href substitutions following the exact Phase 63 D-16 pattern (`${APP_ROUTES.import}?fileId=...`
→ `importFileDetailHref(...)`) — no new logic, no new branching, same call-site shape in all
three files. No split required.
</objective>
