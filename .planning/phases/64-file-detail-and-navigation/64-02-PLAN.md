<objective>
Restructure `TransactionTitleEdit` and `ExpenseTitleEdit` so the title text itself becomes a
`Link` to the row's detail page (D-04), while the pencil icon becomes an independent
click target that still opens the existing inline edit form. Today both components wrap the
whole title+pencil pair in a single `<button onClick>` that only ever opens edit mode —
there is no click-through to `/transactions/[id]` or `/expenses/[id]` from the table.

Purpose: DET-09 requires "row-title click navigates to the detail page ... Pencil icon stays
the inline-edit affordance (text = link, pencil = edit — same coexistence as today's file-name
link)" (64-CONTEXT.md D-04). This is the missing piece — Phase 63 built the Dettagli menu
entries but never touched the title-click behavior itself.

Output: updated `TransactionTitleEdit`/`ExpenseTitleEdit` where non-edit-mode renders a
`<Link>` (title text) sitting beside a separate pencil `<button>` (edit trigger); edit-mode
form behavior is unchanged.
</objective>
