## Spacing Scale

Standard 8-point scale ‚Äî all existing components use Tailwind spacing multiples of 4px.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline chip padding, `gap-1` |
| sm | 8px | Compact element spacing, card inner padding, `gap-2` / `p-2` |
| md | 16px | Default element spacing, card padding `p-4` |
| lg | 24px | Section padding, between KPI row and chart, `gap-6` |
| xl | 32px | Layout gaps, page section breaks |
| 2xl | 48px | Major vertical section breaks |
| 3xl | 64px | Page-level top/bottom padding |

Exceptions:
- KPI cards: `min-h-32` (128px) ‚Äî existing constraint from `kpi-card-reading.tsx`, retained as-is.
- Chart height: `h-[260px]` ‚Äî existing constraint from `overview-chart.tsx`, retained for the 3-bar layout.
- Touch targets on bar click zones: no explicit change ‚Äî bars are full-height within 260px container.

## Chart Contract

### OverviewChart ‚Äî 3-bar grouped layout

| Property | Value |
|----------|-------|
| Chart type | Grouped `BarChart` (Recharts), `barGap={4}`, `barCategoryGap="24%"` |
| Bar order per month group | 1. Entrate (green `--total-in`), 2. Uscite (orange `--total-out`), 3. Accantonato (purple `--total-allocation`) |
| Zero-amount bar | Always rendered, zero-height ‚Äî do NOT hide on empty months (D-01) |
| Bar fill opacity | Selected month: `1.0` / all others: `0.4` ‚Äî same pattern as existing entrate/uscite |
| Label above bar | Compact k-notation (`formatEurCompact`), `fontSize={10}`, `fill-muted-foreground` |
| On bar click | `onMonthSelect(index, direction)` ‚Äî parent updates `selectedMonth` AND `selectedDirection` |
| Tooltip | Extended `NatureTooltip` ‚Äî adds Accantonamento section (Risparmio + Investimento rows) |
| Chart height | `h-[260px]` ‚Äî unchanged |
| Zero bar click | Clicking a zero-height allocation bar selects that month/direction; panel shows empty state |

### Allocation movers panel layout

| Property | Value |
|----------|-------|
| Heading | "Accantonamenti di {mese} rispetto a {mese precedente}" |
| Row grain | Per nature: "Risparmio" / "Investimento" (NOT per category ‚Äî D-03) |
| Row format | Nature name + `formatEur(abs(delta))` + signed Œî indicator |
| Positive Œî sign | `+‚Ç¨N` in `text-[var(--total-in)]` (more allocated = good) |
| Negative Œî sign | `-‚Ç¨N` in `text-[var(--total-out)]` |
| Empty state | "Nessun accantonamento in {mese}." (see Copywriting Contract) |
| Max rows | 2 (only 2 allocation natures: savings + investment) |

## Accessibility Notes

- Allocation bar: aria is handled by Recharts `<Bar>` ‚Äî no additional aria label needed beyond existing chart pattern.

- Direction filter rename: update `<label>` text from "Tipo" to "Direzione" for screen readers.
