## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, delta badge padding |
| sm | 8px | Card inner padding compact, badge gap |
| md | 16px | Card content padding (`p-4`), element spacing |
| lg | 24px | Gap between dashboard sections (`gap-6`) |
| xl | 32px | Page horizontal padding |
| 2xl | 48px | Major section breaks (KPI row → breakdown) |
| 3xl | 64px | Page-level top padding |

Exceptions:
- KPI card touch target on mobile: minimum 44px height (accessibility, iOS tap target)
- Chart container minimum height: 300px (Recharts layout constraint)
- YAxis label column in horizontal bar chart: 110px left margin (`margin={{ left: 120 }}`)

### Accent Reserved For

Accent (`--primary` / emerald-600) is reserved for these specific elements ONLY:
1. totalIn (entrate) KPI card value text
2. entrate series bars in MonthlyTrendChart
3. positive delta badge text (entrate went up, savingsRate positive)
4. uncategorizedCount badge in sidebar (Phase 1 D-02 — "invito all'azione")
5. primary/active state in tab filter (Uscite tab border/text when selected)

### KPI Card Color Assignment (D-06)

| KPI | Value color | Delta positive color | Delta negative color |
|-----|-------------|---------------------|---------------------|
| totalIn | `text-emerald-600` (`--total-in`) | `text-emerald-600` | `text-red-500` |
| totalOut | `text-red-500` (`--total-out`) | `text-red-500` (more = worse) | `text-emerald-600` (less = better) |
| balance | `text-slate-700` (`--balance`) | `text-emerald-600` | `text-red-500` |
| savingsRate | conditional: emerald if >= 0, red if < 0 | `text-emerald-600` | `text-red-500` |
| uncategorizedCount | `text-foreground` (neutral) | `text-red-500` (more = worse) | `text-emerald-600` (less = better) |

Note: `deltaIsGoodWhenPositive = true` for totalIn, balance, savingsRate. `deltaIsGoodWhenPositive = false` for totalOut, uncategorizedCount.

### Chart Series Colors (D-07, chartConfig)

```typescript
const trendChartConfig = {
  totalIn:  { label: 'Entrate',           color: 'var(--total-in)'  },  // emerald-600
  totalOut: { label: 'Uscite',            color: 'var(--total-out)' },  // red-500
  totalNc:  { label: 'Non categorizzato', color: 'var(--muted-foreground)' },  // slate-500
  totalIgn: { label: 'Ignorato',          color: 'var(--muted)' },     // slate-100

```

For breakdown chart, use `var(--total-out)` for uscite bars (default tab) and `var(--total-in)` for entrate bars.

## Layout Contract

### KPI Cards Row (D-04, D-05)

Desktop (>= 768px):
```
grid grid-cols-5 gap-4
```
5 columns in a single row. Each card: equal width, flex-col layout, p-4 padding.

Mobile (< 768px):
```
grid grid-cols-2 gap-3
```
5 cards in 2+2+1 pattern. Last card centered:
```jsx
// 5th card spans full width and is centered:
<div className="col-span-2 flex justify-center">
  <KpiCard className="w-1/2" ... />   {/* or max-w-[50%] */}
</div>
```
No horizontal scroll on any viewport.

### KPI Card Internal Structure

```
Card (p-4, flex-col, gap-1)
├── Label     — 12px, muted-foreground, weight 400
├── Value     — 24px Geist Mono, weight 600, color per KPI table above
└── Badge     — variant="outline", 12px, delta text, color per KPI table above
```

Delta text rules:
- `null` → render em dash `—` in muted-foreground color
- `0` → render `0%` in muted-foreground color
- `> 0` → render `+{N}%` (e.g. `+12%`)
- `< 0` → render `{N}%` (e.g. `-8%`, minus sign is part of the number)

Delta badge does NOT use an arrow icon — plain text `+/-N%` only (Claude's discretion applied: simpler, less visual noise).

### Filter Toolbar (D-03)

Positioned above the breakdown chart section:
```
Flex row, gap-2, items-center, pb-4
├── Tabs (Uscite | Entrate | Tutti) — default: Uscite
└── Select (date preset)            — default: last-month
    Options: Ultimo mese | Ultimi 3 mesi | Ultimi 6 mesi | Quest'anno | Anno scorso
```

Tabs use shadcn `Tabs` component with `variant="outline"` style. The active tab has `--primary` border and text color. Inactive tabs use muted-foreground.

URL params written on change:
- `?type=out` (default) | `?type=in` | `?type=all`
- `?preset=last-month` (default) | `?preset=last-3-months` | `?preset=last-6-months` | `?preset=this-year` | `?preset=last-year`

### Category Breakdown Chart (D-02)

```
Section wrapper: Card with CardHeader ("Breakdown Categorie") + CardContent
├── DashboardFilters (toolbar above chart)
└── CategoryBreakdownChart ('use client')
    ├── ChartContainer — min-h-[300px] w-full
    └── BarChart layout="vertical" margin={{ left: 120 }}
        ├── XAxis type="number" domain={[0, 100]} unit="%" hide
        ├── YAxis dataKey="name" type="category" width=110 tick fontSize=12
        │   onClick → toggles category expansion (shows subcategory rows)
        └── Bar dataKey="pct" fill="var(--total-out)" radius={[0, 4, 4, 0]}
            └── Cell for subcategory rows: fill="var(--muted)" (lighter shade)
```

Drill-down behavior (Claude's discretion — static, no animation):
- Category rows are clickable (cursor-pointer on YAxis tick).
- Click expands subcategory rows inline below the parent in the flat data array.
- Expanded subcategory bars use a lighter fill (`var(--muted-foreground)` at 0.4 opacity, or `text-slate-300`).
- Maximum 15 categories shown. If > 15, show top 15 by count with an "Altri N categorie" summary row.
- No pagination UI — cutoff at 15 is silent (documented behavior for v1).

### Monthly Trend Chart (D-07, D-08)

```
Section wrapper: Card with CardHeader ("Trend Mensile") + CardContent
└── MonthlyTrendChart ('use client')
    ├── ChartContainer — min-h-[300px] w-full
    └── BarChart data={trend} barGap={2} barCategoryGap="20%"
        ├── XAxis dataKey="month"        — short format "Apr", "Mag" (locale it-IT)
        ├── YAxis                        — auto domain
        ├── ChartTooltip content={<ChartTooltipContent />}
        ├── ChartLegend content={<ChartLegendContent />} onClick={toggleSeries}
        ├── Bar dataKey="totalIn"  fill="var(--total-in)"  hide={hidden.has('totalIn')}
        ├── Bar dataKey="totalOut" fill="var(--total-out)" hide={hidden.has('totalOut')}
        ├── Bar dataKey="totalNc"  fill="var(--muted-foreground)" hide={hidden.has('totalNc')}
        └── Bar dataKey="totalIgn" fill="var(--muted)" hide={hidden.has('totalIgn')}
```

Legend toggle state: `useState<Set<SeriesKey>>(new Set())`. Default: all 4 series visible.

## Interaction Contract

### Filter Changes

| Interaction | Trigger | Effect | URL mutation |
|-------------|---------|--------|-------------|
| Tab click (Uscite/Entrate/Tutti) | click | Re-renders BreakdownSection with new type | `?type={out|in|all}` |
| Preset Select change | onChange | Re-renders BreakdownSection + TrendSection with new range | `?preset={value}` |
| Category bar click | YAxis tick onClick | Toggles drill-down (local state, no URL change) | none |
| Legend item click | ChartLegend onClick | Toggles series hide (local state) | none |

Filter changes use `useTransition` + `router.replace` with `{ scroll: false }`. No full page reload.

### Empty / Zero States

Phase 4 operates on manual expenses only. Monetary amounts (totalIn, totalOut, balance, savingsRate) will be `0` until Phase 5. This is expected behavior — the UI handles it gracefully:

| Condition | Display behavior |
|-----------|-----------------|
| All monetary KPIs = 0 | Show `€0,00` in Geist Mono. Delta shows `—` (null sentinel). No special empty state UI. |
| uncategorizedCount = 0 | Show `0` with delta `—`. Normal display. |
| No expenses in period | Breakdown chart: show empty state card (see copy below). Trend chart: show empty state card. |
| Breakdown has data but type filter returns 0 | Show empty state card for selected tab. |

## Number Formatting

All monetary values are formatted for Italian locale:

| Value type | Format | Example |
|-----------|--------|---------|
| Monetary (Phase 5+) | `€` + thousands dot + decimal comma | €1.234,56 |
| Monetary Phase 4 (always 0) | €0,00 | €0,00 |
| Savings rate | N% or N,N% | 23% or 23,4% |
| Count | integer, no separator | 42 |
| Delta percentage | +N% or -N% or — | +12% / -3% / — |
| Chart axis (monetary) | abbreviated if >= 1000: €1.2k | €1.2k |
| Chart axis (count) | integer | 42 |

Use `Intl.NumberFormat('it-IT', { style: 'currency', currency: 'EUR' })` for formatting. Do NOT use JS native `+` on amounts — all computation must use Decimal.js before formatting.

## Accessibility

| Requirement | Implementation |
|-------------|---------------|
| KPI cards keyboard navigable | Cards are static display — no interactive elements inside except Badge |
| Filter tabs keyboard navigable | shadcn Tabs uses Radix UI — keyboard navigation built-in (arrow keys) |
| Chart interactivity | YAxis tick onClick provides drill-down; add `role="button"` and `tabIndex={0}` to clickable ticks |
| Legend toggle | ChartLegend items are clickable — shadcn ChartLegendContent handles focus |
| Color-only data differentiation | Each chart series is identified by both color AND label (tooltip + legend text) |
| Minimum touch target | 44px height for KPI cards on mobile |
| Null delta semantics | `—` rendered as text (readable by screen readers) |
