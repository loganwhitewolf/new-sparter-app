```typescript
type Props = {
  label: string
  value: string
  delta: number | null
  tone: 'in' | 'out' | 'balance' | 'savings' | 'neutral'
  goodWhenPositive?: boolean
  className?: string
}
```

Rules:
- Render `Card` + `CardContent`.
- Label class: `text-xs text-muted-foreground`.
- Value class includes `font-mono text-2xl font-semibold`.
- Value color:
  - `in` -> `text-total-in`
  - `out` -> `text-total-out`
  - `balance` -> `text-balance`
  - `savings` -> emerald when value does not start with `-`, red when it does
  - `neutral` -> `text-foreground`
- Delta badge text:
  - `null` -> `--`
  - `0` -> `0%`
  - positive -> `+{N}%`
  - negative -> `{N}%`
- Delta badge color uses `goodWhenPositive`; for `totalOut` and `uncategorizedCount`, negative deltas are good.
- Add `data-testid` as `kpi-${label.toLowerCase().replaceAll(' ', '-')}`.

Create `components/dashboard/kpi-cards.tsx` as a Server Component.

It renders five cards in this order:
1. `Totale entrate` value `data.totalIn`, tone `in`, delta `data.deltas.totalIn`, goodWhenPositive true
2. `Totale uscite` value `data.totalOut`, tone `out`, delta `data.deltas.totalOut`, goodWhenPositive false
3. `Bilancio` value `data.balance`, tone `balance`, delta `data.deltas.balance`, goodWhenPositive true
4. `Tasso risparmio` value `${data.savingsRate}%`, tone `savings`, delta `data.deltas.savingsRate`, goodWhenPositive true
5. `Da categorizzare` value `String(data.uncategorizedCount)`, tone `neutral`, delta `data.deltas.uncategorizedCount`, goodWhenPositive false

Grid classes:
- Wrapper: `grid grid-cols-2 gap-3 md:grid-cols-5 md:gap-4`
- Fifth item wrapper: `col-span-2 flex justify-center md:col-span-1 md:block`
- Fifth card width on mobile: `w-1/2 min-w-36 md:w-auto`
