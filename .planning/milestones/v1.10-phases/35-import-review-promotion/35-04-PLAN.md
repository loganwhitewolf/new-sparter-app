## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-35-04 | Information Disclosure | AnalyzePage → ImportPreview (categories prop) | accept | `getCategories()` returns categories scoped to the session user (per `lib/dal/categories.ts` — `verifySession` inside). Sending these to the client is safe because the user already owns/sees them in `/settings/categories`. |
| T-35-05 | Denial of Service (slow page) | AnalyzePage parallel fetch | mitigate | `Promise.all` runs `analyzeImportAction` and `getCategories` concurrently; `getCategories` is React-cache wrapped so concurrent invocations within the same request are deduplicated. Page latency is bounded by the slower of the two. |
| T-35-06 | Tampering (suggestion section bypass) | ImportPreview confirm flow | mitigate | The confirm block is rendered independently of `SuggestionSection`; no shared state. REV-04 contract is verified by `tests/import-preview-ui.test.tsx` (the REV-04 test asserts both elements coexist). |
