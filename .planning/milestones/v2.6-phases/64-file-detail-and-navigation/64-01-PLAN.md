|-----------|----------|-----------|----------|-------------|-----------------|
| T-64-01 | Information Disclosure | getTransactionsByFileId | high | mitigate | WHERE clause requires both `transaction.fileId = :fileId` AND `transaction.userId = :userId` — a guessed/enumerated fileId belonging to another user returns an empty array, never another user's rows |
| T-64-02 | Tampering | importFileDetailHref | low | accept | Pure string builder, no I/O; encodeURIComponent prevents path injection via a malicious fileId value reflected into an href |
