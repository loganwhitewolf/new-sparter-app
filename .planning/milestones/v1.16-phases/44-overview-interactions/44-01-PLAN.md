<objective>
Establish the Wave 0 test scaffold for Phase 44 and extract the pure, filter-aware chart reduction helpers that the income/expense filter chips will consume in Plan 03.

Purpose: The current `deriveBarRow` is private inside `overview-chart.tsx` and sums ALL buckets unconditionally. Phase 44 needs a version that includes only the selected buckets. Extracting it into a pure, co-located utility makes filter slicing unit-testable (FILT-01/02/03) without rendering Recharts, and gives Plans 02–03 a stable test file to append to. KPI independence (FILT-03) is provable at the unit level because the helper only touches chart buckets, never KPI totals.

Output: `components/dashboard/overview/overview-chart-utils.ts` (pure helpers + key constants) and `tests/overview-interactions.test.tsx` (the shared Phase 44 test file, seeded with filter-reduction tests).
</objective>
