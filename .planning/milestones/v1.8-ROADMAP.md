# Milestone v1.8: Dashboard Intelligence

**Status:** ✅ SHIPPED 2026-05-20
**Phases:** 29
**Total Plans:** 4

## Overview

M008 adds deviation analysis and chart clarity to the Sparter dashboard. The deviation view shows each category's spend vs a 3-month baseline of the previous calendar month, surfaced as colored percentage badges on the categories list and detail pages. The MonthlyTrendChart is replaced by two focused charts: EntrateUsciteChart (Entrate/Uscite bars) and BilancioBarsChart (per-month green/red Bilancio bars). A sort toggle on the categories page lets users rank by deviation magnitude or total amount.

## Milestone Summary

**Key Decisions:**
- D-01: Fix last-month date preset — both `from` and `to` use `month - 1`
- D-02: Reference Period = hardcoded last completed calendar month (ignores caller preset)
- D-03: Baseline window = 3 calendar months immediately preceding the Reference Period
- D-04: Deviation formula = `(ref - base) / abs(base) * 100` — signed percentage
- D-05: Noise threshold = €15 — categories below excluded from deviation display
- D-06/D-08: Both category pages get deviation data; D-09: display as % only (no euro delta)
- D-07: Sort default = deviation; URL omits sort param when it matches the default
- D-09: Color polarity — `out` positive = red (overspend), `in` positive = green (income growth)
- D-10: MonthlyTrendChart fully deleted — replaced by two single-purpose charts
- D-11: EntrateUsciteChart — 2 bars only (totalIn/totalOut)
- D-12: BilancioBarsChart — per-month `<Cell>` colored by balance sign
